class dafi.dyn_model.DynModel(nsamples, da_interval, t_end, max_da_iteration, input_file)[source]¶

Parent class for dynamic models.

Use this as a template to write new dynamic models. The required attributes and methods are summarized below.

name¶
Type

Name of the forward model for reporting. str

nstate¶

Number of states in the state vector. int

Number of states in the observation vector. int

Initial mean value of the state vector. ndarray
